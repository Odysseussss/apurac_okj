# Visão Geral do Projeto - Portal de Apuração

## Propósito do Projeto
O **Portal de Apuração** é uma ferramenta web desenvolvida para facilitar o trabalho de analistas na apuração de campanhas de incentivo. Ele permite que qualquer pessoa interessada inicie um processo de apuração de forma intuitiva, eliminando a dependência de processos manuais complexos no Excel.

## Funcionalidades Principais
- **Importação de Dados**: Suporte para carregamento de planilhas de Produtos e Vendas (`.xlsx`, `.csv`).
- **Navegação Inteligente**: Sidebar lateral para alternar entre o painel de apuração e o histórico.
- **Mecânicas de Apuração**:
  - **Mínimo**: Baseado no volume total.
  - **Escalonada**: Baseado em categorias específicas (ex: Bebidas, Mercearia).
  - **Combo**: Recompensa para compra casada de produtos específicos.
- **Histórico Local**: Armazenamento seguro no navegador das apurações realizadas.
- **Ranking Enterprise**: Visualização de alta densidade com detalhamento por cliente.

## Público-Alvo
- Analistas de Vendas.
- Gestores de Campanhas.
- Equipes de Incentivo.

## Benefícios
- **Agilidade**: Redução no tempo de processamento de grandes volumes de dados de vendas.
- **Precisão**: Garantia de que as regras da mecânica sejam aplicadas de forma consistente.
- **Transparência**: Visualização clara de quem atingiu as metas e o valor da premiação devida.
