# Release Notes - Versão 0.2

A versão 0.2 transforma o portal em uma ferramenta completa de gestão com navegação estruturada e suporte a mecânicas complexas.

## Novidades
- **Sidebar de Navegação**: Alternância rápida entre o painel de apuração e o histórico.
- **Mecânica Escalonada**: Configuração de metas e prêmios por **Categoria de Produto**.
- **Mecânica Combo**: Bonificação para compra de conjuntos específicos de SKUs.
- **Meu Histórico**: Sistema de salvamento local (`localStorage`) para consultas rápidas.
- **UI Pro**: Interface refinada com suporte a Tailwind CSS v4 e Lucide Icons.

## Histórico de Versões
- **Versão 0.1**: Lançamento inicial com painel de apuração e mecânica de ranking.
- **Versão 0.2**: Adição de mecânicas escalonadas e combo, histórico local e UI refinada.
